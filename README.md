# Survival Game

Survival-Game is a simple game with 2D layout. Players must kill other entities and health must be greater than 0 to win the game, otherwise the game will be lost.

## Entities and the attributes

### 1. **Player**

    aliases: P
    health: 100
    power: 100

### 2. **Zombie**

    aliases: Z
    health = 50
    power = 10

### 3. **Skeleton**

    aliases: S
    health = 60
    power = 20

### 4. **Creeper**

    aliases: Z
    health = 70
    power = 30
